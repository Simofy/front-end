API end points

Board

Get array of cells for specified rectangle, [x, x + w) & [y, y + h)

GET /api/board
Params

x - origin of board

y - origin of board

w - width of board

h - height of board

Response
Array of:

x - position of cell

y - position of cell

data - last history of cell

name - name of user that

color - color

createdAt - last time cell was updated

Response example
[
  {
    x: 5,
    y: 7,
    data: {
      name: "Alio valio",
      color: "red",
      createdAt: "2021-03-31T19:27:28.329Z"
    }
  }, ...

Update Board

Update one cell

POST /api/board
Body

x - origin of cell

y - origin of cell

name - name of author

color - color for cell

Cell

Get info about cell

GET /api/board/cell
Params

x - origin of cell

y - origin of cell

Response
Array of:

x - position of cell

y - position of cell

history - array of cell changes

name - name of user

color - color

createdAt - time when cell changes where created

Response example
[
  {
    x: 6,
    y: 15,
    history: [
      {
        name: "12Alio valio",
        color: "red",
        createdAt: "2021-03-31T19:29:15.040Z"
      },
      {
        name: "12Alio valio",
        color: "red",
        createdAt: "2021-03-31T19:33:26.296Z"
      }
    ]
  }
]

Status

Get info about board state

GET /api/board/status
Response
Array of:

maxX - largest X position for Cell

maxY - largest X position for Cell

minX - smallest X position for Cell

minY - smallest Y position for Cell

update - how many times board updated

Response example
[
  {
    maxX: 16,
    maxY: 15,
    minX: 5,
    minY: 7,
    update: 7
  }
]

Fetch examples

GET example 

fetch('/api/board?x=0&y=0&w=10&h=10', {
  method: 'GET'
})

POST example 

fetch('/api/board', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    x: 6,
    y: 10,
    name: '12Alio valio',
    color: 'red'
  })
})
